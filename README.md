# Flight Pricing Analysis

## Overview
This project analyzes airline pricing patterns using a dataset of over 300,000 flight records from Goibibo. The analysis focuses on routes between Bangalore and Hyderabad and explores pricing differences, airline distribution, and the impact of ticket class on fares.

## Research Questions
- How do ticket prices differ between Bangalore–Hyderabad and Hyderabad–Bangalore routes?
- How do prices vary across airlines?
- How does business class affect airline pricing comparisons?
- Which destinations are most common from Bangalore?

## Tools Used
- R
- readxl
- ggplot2

## Key Findings
- Flights from Bangalore to Hyderabad had a higher median ticket price than the reverse route.
- Vistara and Air India appeared significantly more expensive due to business-class offerings.
- After controlling for ticket class, price differences between airlines became much smaller.
